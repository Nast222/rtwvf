# Movie Library — личная кинотека

Автор: [Кецма]

Описание
Это простое и удобное графическое приложение для ведения личной коллекции фильмов. Оно позволяет добавлять, просматривать и фильтровать фильмы по жанру и году выпуска. Все данные надёжно сохраняются в файл, поэтому ваша коллекция не потеряется после закрытия программы.

Технологии
Язык: Python 3.x.
Интерфейс: Tkinter (входит в стандартную библиотеку).
Хранилище данных: JSON (файл movies.json).
Контроль версий: Git.
Как запустить
Убедитесь, что у вас установлен Python 3.
Скачайте или клонируйте этот репозиторий.
Запустите файл movie_library.py из терминала:
bash
Копировать
python movie_library.py
Как пользоваться
Интерфейс программы разделён на две вкладки для удобства.

Вкладка «➕ Добавить фильм»
Здесь вы можете пополнить свою коллекцию:

Заполните все поля: Название, Жанр, Год и Рейтинг.
Нажмите кнопку «Добавить фильм».
Новый фильм мгновенно появится в таблице ниже.
⚠️ Программа проверяет ввод: Год должен быть числом, а рейтинг — от 0 до 10.

Вкладка «🔎 Фильтр»
Используйте эту вкладку для быстрого поиска:

Введите интересующий вас жанр (например, «Комедия») или год (например, «2010») в соответствующие поля.
Нажмите кнопку «Фильтровать».
В таблице отобразятся только те фильмы, которые соответствуют вашему запросу.
Сохранение данных
Вам не нужно беспокоиться о сохранении. Приложение автоматически записывает все изменения в файл movies.json. При следующем запуске вы увидите свою коллекцию в том же виде, в каком оставили её ранее.

Примеры тестов (Проверка работы)
Вы можете проверить корректность работы программы, выполнив следующие действия:

Тест на некорректный ввод:
Попробуйте ввести в поле «Год» буквы (например, «две тысячи»). Программа покажет предупреждение: «Год должен быть числом от 1888 до 2100!».
Попробуйте ввести рейтинг «11». Программа покажет ошибку: «Рейтинг должен быть числом от 0 до 10!».
Тест на фильтрацию:
Добавьте несколько фильмов разных жанров и лет.
Введите в поле фильтрации жанр одного из добавленных фильмов. В таблице должен остаться только он.
Тест на сохранение:
Добавьте фильм, закройте программу и запустите её снова. Добавленный ранее фильм должен быть на месте.
